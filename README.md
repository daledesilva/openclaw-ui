# openclaw-ui

A dedicated PWA web interface for OpenClaw.

## Architecture & Documentation

Please refer to the [Docs folder](docs/) for our architectural decisions, component patterns, and reasoning for the tech stack used.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL provided (usually `http://localhost:5173`).
